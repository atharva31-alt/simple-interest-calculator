# Simple Interest Calculator

A lightweight, efficient script designed to calculate simple interest based on user inputs for Principal, Rate of Interest, and Time Period.

## Formula Used
The application calculates simple interest using the standard mathematical formula:

$$A = P \cdot (1 + r \cdot t)$$

Where:
- **$A$** = Total Accumulated Amount (Principal + Interest)
- **$P$** = Principal Amount
- **$r$** = Annual Interest Rate (as a decimal)
- **$t$** = Time Period (in years)

## Features
- Dynamic input validation for negative values or empty strings.
- Accurate decimal precision outputs.
